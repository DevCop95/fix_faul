# 🔧 fix\_faul · Fix npm Global Install · Windows + Git Bash

Repositorio de documentación y guías para resolver errores comunes al instalar paquetes npm globales en Windows usando Git Bash, especialmente con herramientas AI CLI modernas.

## 📌 Problema que resuelve

Errores al instalar globalmente en Windows con `npx` o `npm install -g`:

- `@google/gemini-cli`
- `@openai/codex`
- `@anthropic-ai/claude-code`
- Otros Node-based CLIs

## ⚡ Inicio rápido

```bash
npx @google/gemini-cli
```
